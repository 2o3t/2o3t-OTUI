将动作或菜单折叠到下拉菜单中。

> Base



## 圆角

> Round

通过 `round` 进行圆角UI的展示.

## 禁用状态

> Disabled

禁用状态

> Size

> Clickable

可通过 `manual` 和 `clickable` 配合手动点击进行操作.
