# InputNumber 计数器

仅允许输入标准的数字值，可定义范围

## 基础用法

> Base



## 圆角

> Round



## 禁用状态

> Disabled



## 大小

> Size

提供了 mini、small、normal、big 四种尺寸.

## 单边调节器

> DoubleRight

可在右侧进行调节.

## 步数与精度

> Step

使用 Step 确定单次步数, precision 确定精度.

## 最大值与最小值

> Limit

可设置上下限: min=最小值, max=最大值.

## 只读状态

> Readonly

数值不可手动编辑.
