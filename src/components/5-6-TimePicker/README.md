# TimePicker 时间选择器

用于选择或输入时间.

## 基础用法

> Base

可以选择任意时间

## 圆角

> Round



## 禁用状态

> Disabled



## 大小

> Size

提供了 `mini`、`small`、`normal`、`big` 四种尺寸.

## 固定时间点

> Options

提供几个固定的时间点供用户选择, 分别通过`star`、`end`和`step`指定可选的起始时间、结束时间和步长.

## 任意时间范围

> Double

可选择任意的时间范围.
