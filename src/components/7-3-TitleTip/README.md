# Title Tip

常用的悬停提示。

## 基础用法

> Base

提供有箭头和无箭头方式

## 插槽用法

> Slots



## 文字排列

> Align

默认文字居中, 提供 left、center、right

## 禁用状态

> Disabled



## 大小

> Size

提供了 mini、small、normal、big 四种尺寸.

## 手动模式

> Manual

提供手动点击模式

## 触控

> Enterable

鼠标是否可进入到内容中

## 图标

> Icon

提供带有图标的弧度感觉的圆角矩形

## 圆形图标

> CircleIcon

提供圆形图标按钮
