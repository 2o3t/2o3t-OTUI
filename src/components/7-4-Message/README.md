# Message 消息提示

常用于主动操作后的反馈提示。与 Notification 的区别是后者更多用于系统级通知的被动提醒。

## 基础用法

> Base

从顶部出现，3 秒后自动消失。

## 圆角或边框

> RoundBorder

可配置消息框为圆角, 或带有边框样式

## 位置

> Position

提供三个位置的显示方式 `left`, `right`, `center`

## 大小

> Size

提供了 `mini`、`small`、`normal`、`big` 四种尺寸.

## 圆形矩形

> Circle

提供弧度感觉的圆角矩形

## 图标

> Icon

提供带有图标的弧度感觉的圆角矩形

## 颜色

> Colors

根据颜色名称, 变化样式
