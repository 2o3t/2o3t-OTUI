# Carousel 走马灯

在有限空间内，循环播放同一类型的图片、文字等内容.

## 基础用法

> Base

适用广泛的基础用法

## 指示器

> Indicator

可以将指示器的显示位置设置在容器外部.

## 切换箭头

> Arrow

可以设置切换箭头的显示时机.

## 大小

> Size

提供了 `mini`、`small`、`normal`、`big` 四种尺寸.

## 卡片化

> Card

当页面宽度方向空间空余，但高度方向空间匮乏时，可使用卡片风格.

## 圆角

> Round
